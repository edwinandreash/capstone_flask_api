Api Documentation

Get all Book Data
{{data.url|safe}}/getallbook

This api return all complete book data.

Get all Author Data
{{data.url|safe}}/getallauthor

This api return all names of book author.

Get Book Count of an Actor
{{data.url|safe}}/getauthorbookcount/<author>

This api return the number of books an author has written. (Example of use: {{data.url|safe}}/getauthorbookcount/J.K. Rowling)

Get Books with Minimum Rating
{{data.url|safe}}/getbookminrating/<rating>

This api return the title of books with specified minimum rating. (Example of use: {{data.url|safe}}/getbookminrating/4.5)

Get Author Books Mean Average Rating
{{data.url|safe}}/getaveragebookauthors

This api return authors with mean of average book ratings.
